## Getting Started


### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/fad-anes/FastReact-web.git
   ```

2. Install dependencies:
   ```bash
   yarn install
   ```


3. Start the development server:
   ```bash
   yarn dev
   ```
